# The Atomic Dial
### An adaptation of the [SmartKnob](https://github.com/scottbez1/smartknob) project by Scott Bezek, this projects adapts the original device to work smoothly with the RP2350B microcontroller. The board was designed to include header pins to plug in the Proton development board designed by staff for ECE 36200: Microprocessor Systems and Interfacing at Purdue University.
##### by Yonathan Gur, Ryan Gelston, Ty Bequette, and Cody Fuh
